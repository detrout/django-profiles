=========================
django-profiles changelog
=========================

Version 0.5
-----------

* Django 1.6 Compatibility


Version 0.3
-----------

* Django 1.4 and 1.5 Compatibility

* Updated the username url regex to match Django 1.3 behaviour.

Version 0.2, 6 November 2008:
-----------------------------

* Project hosting moved from Google Code to Bitbucket, and from a
  Subversion repository to Mercurial.

* Django 1.0 compatibility.

* Views now accept an ``extra_context`` argument, identical to the way
  that argument works in Django's generic views.


Version 0.1, 24 November 2007:
------------------------------

* Initial release.
